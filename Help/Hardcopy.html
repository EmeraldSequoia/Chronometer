Printing the Help Pages

If you have a device running iOS 4.2 or later, and a printer that supports AirPrint, you can print each of EMERALD_PRODUCT's web pages directly, via the "Print this page" links at the top and bottom left of the page.

If you are running on a device that is not able to be upgraded to iOS 4.2, in-app printing is not available, but all of the Help is reproduced in a slightly different format but with the same content on our website at:

http://www.emeraldsequoia.com/h http://www.emeraldsequoia.com/g
From there you can simply use a browser to print each page.
